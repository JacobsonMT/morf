MoRFCHiBi_Web

Instructions:

  1. Obtain one protein sequence of desired protein in FASTA format.
  2. Enter the sequence into the input box and press "Submit Job".
    • An example sequence can be shown by clicking "Example Input".
    • Input box can be cleared with "Clear".
  3. The name, size and execution time are shown under headings "Label", "Size", and "Execution Time", respectively.
  4. "Status" indicates whether the input is "Running" or "Pending" and when it is complete, the system will indicate if the input "Failed" or if it is "Complete"
  5. Once complete, the results can be analyzed further through the following:
    • Download: The MoRF potential of each amino acid is numerically indicated; a higher number suggests a greater probability that residue is in a MoRF residue.
    • Graph: A graph of the input region is illustrated; the higher peaks illustrate greater probability of a MoRF. Hovering over the graph indicates the MoRF potential and the amino acid position at that point.
    • Save: A new window with the data from MoRF­CHiBi­Web will remain on the server for 24 hours.
    • Cancel: This will remove or cancel any inputs.
